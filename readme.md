cure
====
Color-Unicoded moRE.exe

## Usage

    CURE.exe [-b] {filename(s)}
    CURE.exe -h
    PROGRAM | CURE.exe [-b]

# Option

* `-b` : Set bold on each line.
* `-h` : Help

## How to build

    cd cure
    make.cmd get
    make.cmd
