vbsfmt.cmd
==========

VBScript のソースの予約語の大文字・小文字を補正します。

    vbsfmt.cmd ソース名…

* 一箇所以上変換がされたソースのファイル名が標準出力に一行1ファイル形式で出力されます。
* 元ファイルは「`~`」を付けたファイル名にバックアップされます。
* 二重引用符で囲まれた部分はコンバートされません。

camelfmt.exe
============

引数で与えられたソース中にある単語を、標準入力から読み込まれたリストを元に
大文字・小文字を補正します。vbsfmt.cmd より呼び出されます。
